# Screenshots

Place your project screenshots in this folder with these names:

- `vault.png` - Screenshot of Vault app
- `newsflow.png` - Screenshot of NewsFlow app
- `teammessage.png` - Screenshot of TeamMessage app
- `thinkclear.png` - Screenshot of ThinkClear app
- `aiq.png` - Screenshot of AIQ Beta app

Recommended size: 800x500 pixels or similar aspect ratio (16:10)
